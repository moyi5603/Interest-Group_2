## 一、场景体系（v1.2）

| 场景 | Intent ID | 说明 |
|------|-----------|------|
| A | recommend | 推荐活动 Top 3 |
| B | search_activity | 搜索**活动**（学/做/参加/看/教） |
| C | search_group | 搜索**小组**（群/组/圈子/社团） |
| D | filter | 按时间/类别筛选活动 |
| E | trending | 热门排行 Top 5（含趋势标签） |
| F | detail | 活动或小组详情 |
| G | guide | 兜底引导（置信度 < 0.6） |

**v1.2 核心变化**

- 统一 Mock 意图层 `detectIntent()`，模拟 LLM 意图识别 + 置信度
- B/C/D 场景增加 **无匹配通用回复**（先「暂未找到」→ 行动引导 → 热门/活跃兜底 → 疑问句收尾）
- 回复正文统一使用分隔线 `━━━━━━━━━━━━━━━━━━━━━━━━`
- 活动 vs 小组：活动强调「学/做/参加/看/教」，小组强调「群/组/圈子/社团/加入」

## 四、场景 A · 推荐活动（recommend）

**触发**：推荐 / 好玩的 / 适合我 / 有什么活动（intent=recommend，confidence≥0.6）

根据你的兴趣和近期参与记录，这 3 个活动值得看看：

感兴趣哪个？点卡片查看详情或直接报名 👇

**卡片**：a1、a22、a5

## 五、场景 B · 搜索活动（search_activity）

**触发**：有没有…活动 / 教…的 / 找摄影相关的 / 搜一下讲座

**有匹配**：`🔍 搜索活动「{keyword}」` + 分隔线 + 列表项（小组·分类、时间、简介、报名状态）+ 卡片

**无匹配（v1.2 §4.4）**：

```
🔍 搜索活动「XXX」
━━━━━━━━━━━━━━━━━━━━━━━━
暂未找到与「XXX」相关的活动

💡 你可以试试：
· 换个关键词搜
· 看看当前热门活动 👇

🎯 为你推荐热门活动：（Top 3）
━━━━━━━━━━━━━━━━━━━━━━━━

对热门活动感兴趣，还是想换个词再搜？🔍
```

| 关键词 | 卡片 |

| 羽毛球 | a5 |
| 摄影 | a16 |
| 编程 / Go | a13, a14 |
| 户外 / 徒步 | a8, a2 |

## 七、场景 D · 按条件筛选（filter）

**触发**：下周 / 本周 / 周末 / 近期 / 下个月 / 职场+活动

**有匹配**：`📅 筛选结果：{条件}` + 分隔线 + 活动列表 + CTA

**无匹配（v1.2 §6.4）**：如「下个月有什么活动」→ 热门 Top 3 兜底 + 调整条件引导

| 条件 | 卡片 | 备注 |
|------|------|------|
| 本周 | a3, a1, a5 | Chip 短文案「为你推荐以下活动」 |
| 本周末 | a2, a22 | |
| 下周 | a4, a14 | 基准日 6/3 周一，a2 属本周日不在下周 |
| 下周 · 篮球 | 无匹配 → 热门兜底 | mock 无篮球活动，不再误推 a5 羽毛球 |

| 下个月 | — | 无匹配 → 热门兜底 |

## 十一、原型意图路由（Mock）

```
用户输入 → detectIntent() → confidence < 0.6 ? G : 按 intent 分发
```

| intent | 处理函数 |
|--------|----------|
| detail | 实体关键词 → 详情文案 |
| trending | aiHotActs(5) + 趋势标签 |
| search_group | resolveGroupSearch → formatGroupSearchReply |
| **recommend** | **Top 3 活动 · 短文案 + 卡片（优先于 search_activity，避免「推荐…活动」误路由）** |
| search_activity | resolveActivitySearch → formatActivitySearchReply |
| filter | resolveFilter → formatFilterReply |
| unknown | AI_GUIDE |
