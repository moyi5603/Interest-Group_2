## 1. 产品概述

### 1.1 产品定位

**功能名称**：兴趣小组 AI 助手
**产品类型**：企业级 AI 对话助手
**核心价值**：通过自然语言交互，帮助员工快速发现小组、查询活动、管理兴趣
**目标用户**：全体员工

### 1.2 核心能力矩阵

| 能力ID | 能力名称 | MVP状态 | 说明 |
|--------|---------|--------|------|
| A | 智能推荐 | MVP | 规则打分 + 理由文案 |
| C | 内容生成 | MVP | 活动文案一次生成3条 |
| D | 对话查询 | MVP | 意图解析 → 返回卡片列表 |
| E | 搭子匹配 | 二期 | 找人一起参加活动 |


#### 意图识别逻辑（mock）

```javascript
// 关键词匹配规则（优先级从高到低）
const intentRules = [
  {
    intent: 'cancel_enrollment',
    patterns: ['取消报名', '不想参加', '退 出', '取消'],
    context: 'user_has_enrollment'
  },
  {
    intent: 'recommend_group',
    patterns: ['推荐.*小组', '有没有.*小组', '想加入', '找.*组'],
    requireMatch: 1
  },
  {
    intent: 'list_activity',
    patterns: ['有什么活动', '这周.*活动', '下周.*活动', '活动.*'],
    requireMatch: 1
  },
  {
    intent: 'group_detail',
    patterns: ['(.*)社', '(.*)俱乐部', '(.*)队', '小组介绍'],
    requireMatch: 1,
    extract: 'groupName'  // 提取小组名称
  },
  {
    intent: 'create_hint',
    patterns: ['怎么发起', '怎么创建', '创建.*', '发起.*'],
    requireMatch: 1
  },
  {
    intent: 'my_groups',
    patterns: ['我加入了', '我的小组', '我参加了'],
    requireMatch: 1
  },
  {
    intent: 'interest_tags',
    patterns: ['换.*兴趣', '改.*标签', '兴趣.*', '标签.*'],
    requireMatch: 1
  }
];

function recognizeIntent(message) {
  for (const rule of intentRules) {
    const matchCount = rule.patterns.filter(p => 
      new RegExp(p).test(message)
    ).length;
    
    if (matchCount >= rule.requireMatch) {
      return {
        intent: rule.intent,
        extract: rule.extract ? extractEntity(message, rule.extract) : null
      };
    }
  }
  
  return { intent: 'unknown', extract: null };
}
```
